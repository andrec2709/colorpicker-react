
## About

This is a personal study project I'm working on, as a React learning exercise, and to be part of my [developer portfolio](https://andrecarvalho.io).


## Try out

You can try this app online on [Github Pages](https://andrec2709.github.io/colorpicker-react/).


## Installation (for development)

1. Clone the repository:
```
git clone https://github.com/andrec2709/colorpicker-react/
```

2. (inside colorpicker-react/) Install dependencies:
```
npm install
```

3. Then, for development:
```
npm run dev
```
