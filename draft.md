# How will this app work?

It will be a simple colorpicker for RGB color space: You will have separate sliders for R, G and B values which will build up the final color.

## What can you do?

- [x] Modify any of the three sliders to modify the color value;
- [x] Insert any value between 0 and 255 directly for each slider;
- [ ] Insert a HEX value;
- [ ] Copy the HEX value;
- [ ] Copy each R, G, B value individually.

## How will the app look?

- The background of the color picker will represent the actively evaluated color;
- Three horizontal sliders (R, G, B) ranging from 0 through 255;
- Four input text/number fields, being  three the value of each R, G, B slider and the fourth being the HEX value;
- A "copy" icon by the side of each of the 4 fields above;
